# CineReserve (Movie Seat Booking)

A classic Movie Seat Booking application built entirely with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project demonstrates fundamental DOM manipulation skills and logic implementation without relying on any frontend frameworks or libraries.

![CineReserve](https://github.com/user-attachments/assets/d9c61ed4-0f89-44e6-b301-1569f3b39108)


## Key Features

* **DOM Manipulation:** Dynamic seat selection and UI updates using pure JavaScript.
* **Local Storage:** Persists selected seats and movie data even after page refresh.
* **Real-Time Calculation:** Automatically updates total price and seat count based on user selection.
* **Pure CSS Styling:** Custom layout and visual states (selected, occupied, available) without CSS frameworks.
* **Responsive:** Works seamlessly on different screen sizes.

| Category | Technologies |
